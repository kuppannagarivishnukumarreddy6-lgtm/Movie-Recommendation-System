About MovieAI

MovieAI is a Machine Learning based Movie Recommendation System developed using Python, Flask and Scikit-Learn.

Movie Recommendation

Recommends similar movies using Content-Based Filtering and Cosine Similarity.

Machine Learning

Uses Machine Learning techniques to compare movie genres and generate intelligent recommendations.

Technologies Used

  • Python
  • Flask
  • Pandas
  • NumPy
  • Scikit-Learn
  • HTML5
  • CSS3
